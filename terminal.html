Terminal

Uruchomienie terminala w Linuxie: Ctrl+Alt+T.

Polecenia:

  • pwd - wyświetla pełną ścieżkę
  • klawisz Tab - dopełnia nazwy
  • cd Katalog/ - idź do katalogu
  • cd ~ - idź do katalogu domowego
  • cd .. - idź jeden katalog do góry
  • cd ../.. - idź dwa katalogi do góry (itd.)
  • ls - pokaż zawartość katalogu
  • mkdir - utwórz katalog
  • code . - otwiera katalog w Visual Studio Code
  • code otwórz-mnie-w-code - otwiera konkretny plik w Visual Studio Code
  • git --version - pokazuje wersję zainstalowanego Gita
  • git config --global user.name "Jan Nowak" - tworzy konto dla Jana Nowaka
  • git config --global user.email "jannowak@gmail.com" - przypisuje email
  • git config -- get user.name (lub .email) - wyświetla dane właściciela konta
  • ↑ ↓ - podpowiada poprzednio użyte polecenia
  • git init - zakładanie pustego repozytorium - koniecznie w katalogu z plikami projektu
  • ls -a - pokazuje wszystkie, także ukryte pliki (np. pliki Git-a)
  • git status - pokazuje aktualny stan repozytorium, nic nie zmienia
  • git add . - dodaje pliki z bieżącego katalogu i podkatalogów do zarejestrowania ("zacommitowania")
  • git commit -m "name of version, description (in english)" - dodawanie nowego lub kolejnego rejestru/commitu (w terminalu)
  • git commit --amend -m "change description" - zmienia opis ostatniego commitu
  • gitk - dodatek pokazujący historię naszego projektu i zmiany
  • clear - czyści terminal
  • git push - wysyła nowy commit na GitHub
  • git push --force - poprawia nazwę już wysłanego commitu (najnowszego) - ostrożnie!
